# Changelog

## Unreleased - 2026-04-23

### Added

- 新增九阴“环境检查”页面，集中展示窗口识别、分辨率、DPI、管理员权限和输入后端状态。
- 新增 `JiuYinWindowHelper`、坐标映射、截图服务、环境检查服务和输入探针服务。
- 新增九阴阶段 1 IPC 与 preload API：窗口列表、环境检查、区域截图、输入探针和运行状态查询。
- 新增 F10/F12 阶段 1 热键：F10 切换输入验证模式，F12 请求当前动作后停止。
- `NutInputBackend` 接入真实鼠标键盘操作，支持点击、移动、拖拽、按键、组合键和文本输入。

### Safety

- 阶段 1 只做基础环境和输入有效性验证，不启动团练、授业、采集等业务自动化。
- 热键配置会拦截阶段 1 热键冲突，避免覆盖安全控制。
- 截图仅保存到本机应用数据目录中的调试截图文件夹，不上传、不分析账号或聊天等敏感信息。

## 0.1.0 - 2026-04-23

### Changed

- 项目切换为九阴真经专用方向。
- 移除旧游戏的前端入口、IPC 暴露、打包资源、公共资源和业务脚本。
- 新增九阴首页、阶段 0 设置页和九阴资源目录规范。
- 新增 `src/backend/games` 模块边界、`JiuYinOperator` 空实现和 `InputBackend` 抽象。

### Safety

- 阶段 0 不执行任何九阴自动化任务。
- 后续必须先验证窗口识别、截图区域和 `nut-js` 输入有效性，再开发团练、授业等业务功能。
- 不纳入反检测、窗口绑定、最小化后台、内存、封包、驱动或绕过安全软件相关能力。
